# PacketLab

Interactive networking and packet analysis trainer for cybersecurity students.

## What it does

PacketLab lets students explore real captured network traffic (PCAP files) 
through a clean visual interface, get AI-generated plain-English explanations 
of individual packets, and test their understanding through graded challenges.

## Tech stack

- **Backend:** Python, Django, Django REST Framework
- **Frontend:** React, Vite, Tailwind CSS
- **Database:** PostgreSQL
- **Packet parsing:** Scapy
- **AI:** LLM-powered packet explanations and answer grading
